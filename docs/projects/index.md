## 프로젝트 작성 가이드

<!-- TODO: 이 가이드는 작성 요령입니다. 본인이 다 이해했으면 이 섹션은 삭제해도 됩니다. -->

각 프로젝트 상세 페이지는 다음 흐름을 권장합니다.

1. **한 줄 요약 + 핵심 성과** — 가장 먼저 임팩트를 보여줍니다.
2. **개요** — 기간, 인원, 본인 역할(기여 범위를 명확히).
3. **문제 정의** — 어떤 상황·제약이 있었는가.
4. **해결 과정** — 고려한 대안과 *선택한 이유*, 핵심 의사결정.
5. **결과** — 가능하면 정량 수치(응답시간, 처리량, 비용 등).
6. **회고** — 배운 점, 아쉬운 점, 다시 한다면 무엇을 바꿀지.
